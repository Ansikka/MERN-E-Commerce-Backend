MERN-E-Commerce-Backend
MERN-E-Commerce-Backend is the backend of an e-commerce web application built with MongoDB, NodeJS, and ExpressJS.

The frontend of the application is built with ReactJS, Material UI, React-router-dom,ContextAPI. The source code for the frontend can be found at https://github.com/Ansikka/MERN-E-Commerce-Frontend.

Features
User authentication and authorization(JWT)
Admin dashboard for managing products, orders, users and to show statistics
Payemnt Gateway
Mail Service
Forgot Password & Reset Password
Product listing and search
Product details and reviews
Cart management
Order history
